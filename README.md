## 📌 Overview

This project focuses on predicting the likelihood of heart disease using machine learning algorithms. It analyzes medical and clinical features such as age, cholesterol level, blood pressure, chest pain type, and other health indicators to build an accurate predictive model.

The goal is to assist in early detection of heart disease, which can help healthcare professionals make better decisions and potentially save lives.

---

## 🚀 Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA) with visualizations 📊
* Feature selection and engineering
* Multiple ML models implementation:

  * Logistic Regression
  * Random Forest
  * Support Vector Machine (SVM)
* Model evaluation using:

  * Accuracy
  * Precision
  * Recall
  * F1-score
